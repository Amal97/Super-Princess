SuperPrincess is a PacMan like game developed in Java using JavaFX libraries.
Clone repository:
1. Open terminal/Git CMD
2. Go to your desired working location
3. Type "git clone https://github.com/Amal97/Super-Princess-.git" and press Enter
4. Enter your Username and Password if required

Instructions on running the project:
1. Open Eclipse
2. Click File -> Open Projects from File -> Directory
3. Go to your git folder where you cloned and Click OK
4. Click Finish
5.1 Right click on main project folder -> Run As -> Java Application
or
5.2 Expand project files and look for Main.java
5.3 Double click on Main.java
5.4 Click the green Run button
